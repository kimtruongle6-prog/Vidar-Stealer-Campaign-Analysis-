## 📂 Host-Based Artifacts
Evidence of malicious activity found on the victim's local file system and registry:

### **File System Activity**
Vidar targets sensitive browser data at the following locations:
* `%LocalAppData%\Google\Chrome\User Data\Default\Network\Cookies`
* `%AppData%\Local\Microsoft\Windows\INetCookies\ESE\`
* Scanning for `wallet.dat` and cryptocurrency extension folders.

### **Execution Patterns**
* Masquerading as or injecting into `C:\Windows\System32\svchost.exe`.
* Spawning Chrome instances with `--profile-directory="Default"` to bypass session locks.

### **Registry Keys (Persistence/Config)**
* `HKCU\Software\Microsoft\Windows\CurrentVersion\Run`
* Interaction with `HKLM\Software\Microsoft\...` for security configuration tampering.
